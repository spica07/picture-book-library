# 『달님이 이야기』 시리즈 — 그림 생성용 일러스트 프롬프트 가이드

Midjourney, DALL-E, Stable Diffusion 등 AI 이미지 생성 도구용 프롬프트입니다.
밤하늘이 배경인 시리즈라 **어두워도 포근한 색감**이 핵심입니다.

---

## 1. 공통 스타일 프롬프트 (모든 그림 끝에 붙이기)

```
children's picture book illustration, dreamy night sky theme, deep navy and
soft purple gradient background with warm golden moonlight, twinkling stars,
soft pastel glow, kawaii style characters with big sparkling eyes and rosy
cheeks, gentle gouache painting style, cozy and magical atmosphere, never
scary, clean composition for ages 5-7, no text, no letters
```

**네거티브 프롬프트**:
```
realistic, photo, scary, horror, pitch black darkness, complex details,
text, watermark, adult style, harsh shadows
```

💡 밤 장면이 많으므로 `warm golden moonlight`, `never scary`를 꼭 유지하세요.
어둡고 무서운 분위기가 나오면 `cozy bedtime story mood`를 추가하면 좋아요.

## 2. 캐릭터 시트 프롬프트

⭐ 달님이는 **모양이 5번 변하는 한 캐릭터**라서, 얼굴(눈·볼·미소)을 똑같이 유지하는 것이
이 시리즈 일관성의 핵심입니다. 아래 공통 얼굴 묘사를 모든 단계에 복사해 쓰세요.

**달님이 공통 얼굴 (모든 단계에 포함)**:
```
warm golden-yellow glowing body, big gentle sparkling eyes, soft rosy cheeks,
sweet kind smile
```

### 달님이 — 초승달 단계
```
character design sheet, a thin crescent moon character curved like an eyebrow,
warm golden-yellow glowing body, big gentle sparkling eyes, soft rosy cheeks,
sweet kind smile, shy and humble pose, expressions: hiding behind cloud /
surprised / smiling like a smile in the sky
```

### 달님이 — 상현달 단계
```
character design sheet, a half moon character with right side glowing golden

### 달님이 — 보름달 단계
```

character design sheet, a very thin waning crescent moon character curved to

## 5. 장면 프롬프트 작성 공식 (2~5권 직접 만들 때)

```
[해당 단계의 달님이 묘사] + [동작/표정] + [배경/하늘 색] + [구도/분위기]
+ 공통 스타일 프롬프트
```

⏰ **시간대별 하늘색 가이드** (과학적 정확성 + 분위기 통일):
- 초승달 장면 → `purple-orange twilight sky after sunset` (저녁 서쪽)
- 상현달 장면 → `deep navy early night sky` (초저녁 남쪽)
- 보름달 장면 → `rich midnight blue sky` (한밤)
- 하현달·그믐달 장면 → `soft indigo-to-pink dawn sky` (새벽)

**예시 1 — 3권 장면 7 (토끼 길 비추기):**
```
big round full moon character leaning down to shine a silver spotlight beam
on a forest path, tiny baby rabbit looking up with hopeful teary eyes, rich
midnight blue sky, magical rescue moment + (공통 스타일 프롬프트)
```

**예시 2 — 5권 장면 13 (다시 돌아온 초승달):**
```
thin crescent moon character reappearing in purple-orange twilight western
sky, little girl in pajamas waving joyfully from her garden, reunion scene
full of hope + (공통 스타일 프롬프트)
```

## 6. 일관성 유지 꿀팁

1. **달님이 얼굴 묘사 문구를 절대 바꾸지 마세요** — 몸 모양은 권마다 달라도 얼굴이 같아야 "한 캐릭터"로 보입니다.
2. 보라는 전 권에 등장하므로 **노란 달무늬 잠옷 + 별 머리핀**을 고정 키워드로 유지하세요.
3. Midjourney 사용 시: 1권에서 마음에 든 달님이 이미지를 `--cref`로, 그림체는 `--sref`로 고정.
4. 밤 장면이 너무 어둡게 나오면 `warm golden moonlight illuminating the scene, cozy bedtime story mood` 추가.
5. 양면 펼침 장면(3권 장면 1, 통합편 장면 8)은 `wide panoramic composition, 2:1 aspect ratio` 추가.
6. 통합편 부록 '달 관찰 일기' 페이지는 일러스트가 아닌 **워크시트**이므로:
```
cute worksheet template design, 28 empty circles arranged in 4 weekly rows
for drawing moon phases, small moon and star decorations in corners, soft
pastel colors, printable children's activity page
```
