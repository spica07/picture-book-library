# 『도토리 숲 동물 친구들』 — 그림 생성용 일러스트 프롬프트 가이드

Midjourney, DALL-E, Stable Diffusion 등 AI 이미지 생성 도구용 프롬프트입니다.
사계절이 흐르는 따뜻한 숲 마을이 배경이라 **계절감과 포근한 동화 숲 분위기**가 핵심입니다.

---

## 1. 공통 스타일 프롬프트 (모든 그림 끝에 붙이기)

```
children's picture book illustration, cozy enchanted forest village theme, warm
earthy colors with golden light through trees, a big oak tree at the center,
kawaii style woodland animals with big sparkling eyes and rosy cheeks, soft
rounded shapes, gentle gouache painting style, warm storybook atmosphere, clean
composition for ages 5-7, never scary, no text, no letters
```

**네거티브 프롬프트**:
```
realistic, photo, scary, horror, dark gloomy forest, complex details, text,
watermark, adult style, sharp predator features
```

💡 동물을 의인화하되 `wearing tiny clothes, standing friendly` 정도로, 너무 사람처럼은 X.

---

## 2. 계절 가이드 (시리즈에 흐름 주기)

옴니버스지만 계절을 흐르게 하면 시리즈에 리듬이 생겨요.
- 1권 다람쥐(가을) → `autumn forest, falling red and gold leaves`
- 2권 부엉이(밤) → `night forest under a big moon`
- 6권 곰(겨울) → `snowy winter forest, cozy`
- 통합편 운동회(가을) → `clear autumn sky, festive`
- 나머지는 봄·여름의 `fresh green forest, sunny`

## 6. 장면 프롬프트 작성 공식 (2~10권 직접 만들 때)

```
[해당 동물 캐릭터 묘사] + [동작/표정] + [계절·숲 배경] + [구도/분위기]
+ 공통 스타일 프롬프트
```

```
the color-changing chameleon turning the most beautiful rainbow color after
honestly sharing its feelings, surrounded by warm supportive woodland friends,
sunny forest, joyful glowing moment + (공통 스타일 프롬프트)
```

## 7. 일관성 유지 꿀팁

1. **너구리 선생님(둥근 안경 + 조끼)**을 전 권 고정 — 시리즈를 잇는 연결 고리예요.
2. 여우·부엉이처럼 야생에선 포식자인 동물도 `friendly, gentle, not scary`로 — 모두가 한 마을 친구.
3. 계절 가이드(2번 항목)로 권마다 색감을 다르게.
4. 카멜레온 레오는 **감정=색** 규칙(노랑 기쁨, 파랑 슬픔, 빨강 화남, 무지개 솔직)을 지켜야 8권 메시지가 살아요.
5. 의인화 정도: `tiny clothes/accessories, standing on two legs sometimes` 정도로 통일.
6. 양면 펼침(통합편 운동회·동굴)은 `wide panoramic composition, 2:1 aspect ratio`.
