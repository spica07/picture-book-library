## 5. 장면 프롬프트 작성 공식 (2~10권 직접 만들 때)

각 권의 "그림" 묘사를 아래 공식에 넣으면 됩니다.

```
[캐릭터 시트의 캐릭터 묘사] + [동작/표정] + [배경/장소] + [구도/분위기]
+ 공통 스타일 프롬프트
```

**예시 — 2권 장면 4 (이불 속에 숨은 비방울이):**
```
cute teardrop-shaped raindrop character with translucent sky-blue body hiding
under a thick fluffy cloud blanket, only feet peeking out, sulky expression,
cozy cloud bedroom + (공통 스타일 프롬프트)
```

## 6. 일관성 유지 꿀팁

1. **캐릭터 시트를 먼저 생성**하고, 마음에 드는 이미지를 레퍼런스로 저장하세요.
2. Midjourney는 `--cref [캐릭터 이미지 URL]`, `--sref [스타일 이미지 URL]` 옵션으로 캐릭터·스타일을 고정할 수 있어요.
3. 같은 권 안에서는 **같은 시드(seed) 값**을 쓰면 그림체가 비슷하게 유지됩니다.
4. 캐릭터 묘사 문구는 **매 장면 토씨 하나 안 바꾸고 복사**해 쓰는 게 통일성의 핵심이에요.
5. 본문 글자는 그림에 넣지 말고(`no text` 유지), 이미지 생성 후 편집 도구에서 따로 얹는 것이 한글 깨짐을 막는 방법입니다.
6. 양면 펼침 장면(8권 13장면, 특별판 13장면)은 `wide panoramic composition, 2:1 aspect ratio`를 추가하세요.
