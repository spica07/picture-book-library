# 『쿵쿵 공룡 마을 친구들』 — 그림 생성용 일러스트 프롬프트 가이드

Midjourney, DALL-E, Stable Diffusion 등 AI 이미지 생성 도구용 프롬프트입니다.
화산 아래 공룡 마을이 배경이라 **선사시대 자연 + 친근한 공룡 캐릭터**가 핵심입니다.

---

## 1. 공통 스타일 프롬프트 (모든 그림 끝에 붙이기)

```
children's picture book illustration, friendly prehistoric dinosaur village
theme, lush green valley with ferns and giant leaves, a gentle volcano in the
background, warm sunny colors, kawaii style baby dinosaurs with big sparkling
eyes and rosy cheeks, soft rounded chubby shapes, gentle gouache painting
style, cheerful storybook atmosphere, clean composition for ages 5-7, never
scary, no text, no letters
```

**네거티브 프롬프트**:
```
realistic, photo, scary, horror, sharp menacing teeth, blood, violent, dark
gloomy, complex details, text, watermark, adult style
```

💡 핵심: 공룡은 모두 `chubby, rounded, baby-like, friendly, small soft teeth`.
티라노조차 무섭지 않고 귀엽게.

## 5. 장면 프롬프트 작성 공식 (2~10권 직접 만들 때)

```
[해당 공룡 캐릭터 묘사] + [동작/표정] + [선사시대 배경] + [구도/분위기]
```

## 6. 일관성 유지 꿀팁

1. **모든 공룡은 아기처럼 통통하고 둥글게(chubby, rounded, baby-like)** — 5~7세 대상의 핵심. 사납거나 날카롭게 X.
2. 티라노 티노의 이빨은 반드시 `small, rounded, friendly` — 절대 무섭지 않게.
3. 하늘이(하늘색 프테라노돈 + 우편 가방)를 전 권 고정 — 시리즈 연결 고리.
4. 화산은 늘 `gentle, distant, smoking softly` — 통합편 빼고는 위협적이지 않게 배경에만.
5. Midjourney: 1권 티노·하늘이 이미지를 `--cref`로 고정.
6. 양면 펼침(통합편 대피·새 마을 축제)은 `wide panoramic composition, 2:1 aspect ratio`.
7. 9권 모사는 바다 공룡이라 유일하게 물가 배경 — `lakeside, water` 키워드로 차별화.
