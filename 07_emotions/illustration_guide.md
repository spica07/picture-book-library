# 『마음 마을 친구들』 시리즈 — 그림 생성용 일러스트 프롬프트 가이드

Midjourney, DALL-E, Stable Diffusion 등 AI 이미지 생성 도구용 프롬프트입니다.
감정이 주인공인 시리즈라 **색 = 캐릭터 정체성**입니다. 권마다 주인공 감정의 색이
화면의 주조색이 되도록 유지하세요.

## 1. 공통 스타일 프롬프트 (모든 그림 끝에 붙이기)

```
children's picture book illustration, whimsical tiny village of emotions
floating on soft clouds inside a child's heart, glowing lanterns in many
colors, a lighthouse in the center, soft pastel colors, kawaii style
blob-shaped emotion characters with big sparkling eyes and rosy cheeks,
gentle gouache painting style, warm and cozy atmosphere, never scary,
clean composition for ages 5-7, no text, no letters

## 4. 장면 프롬프트 작성 공식

```
[주인공 감정 캐릭터 묘사] + [동작/표정] + [마음 마을 or 현실 배경]
+ [주조색: 해당 감정의 색] + 공통 스타일 프롬프트

## 5. 일관성 유지 꿀팁

1. **감정 = 색**: 각 권의 주인공 색이 그 권 전체의 주조색이 되게 하세요. (1권은 노랑 톤, 2권은 파랑 톤…)
2. 하루의 **민트색 후드 + 가슴의 하트**는 전 권 고정 키워드입니다.
3. 마음 마을 장면엔 항상 **등대**가 배경 어딘가에 보이게 하면 세계관이 통일됩니다.
4. 현실 장면(집·유치원·놀이터)과 마음 마을 장면이 한 권 안에 섞이므로, 현실 장면 구석에
   해당 감정의 색 빛이 은은하게 비치게 하면 두 무대가 자연스럽게 이어집니다.
5. Midjourney 사용 시: 캐릭터 시트를 먼저 만들어 `--cref`로 고정, 그림체는 `--sref`로 고정.
6. 16:9 영상·웹뷰어용이라면 모든 프롬프트에 `wide 16:9 landscape composition` 추가.
