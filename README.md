## 활용 방법
1. 스토리 파일의 본문으로 그림책 텍스트를 구성하세요.
2. illustration_guide.md가 있는 시리즈는 **캐릭터 시트 프롬프트를 먼저 생성**해 레퍼런스로 고정한 뒤, 장면별 프롬프트로 삽화를 만드세요. 새 시리즈는 권별 장면의 `그림` 설명을 바탕으로 가이드를 확장하세요.
3. 글자는 이미지에 넣지 말고(프롬프트에 `no text` 유지), 편집 단계에서 한글 텍스트를 얹으세요.

  자연·과학 확장 (기존 시리즈와 세계관 연결 가능)

  1. 사계절 친구들 🌸
  봄바람이, 벚꽃이, 매미, 단풍이, 첫눈이, 눈사람, 새싹이, 장마, 고드름, 아지랑이 — 각 권마다 "계절은 가도 다시 온다", 기다림, 변화의 즐거움. 날씨 친구들과 자연스럽게    
  크로스오버 가능. 통합편: 『일 년 마을 큰 여행』 (사계절이 릴레이로 바통을 넘기는 이야기)

  2. 별자리 친구들 ⭐
  북두칠성, 오리온, 카시오페이아, 백조자리, 전갈이, 큰곰이, 작은곰이, 거문고자리, 쌍둥이별, 샛별 — 계절별 별자리 + 그리스/우리 별 이야기. 달님이 시리즈의 할머니
  별·샛별과 같은 세계관이라 시너지가 가장 큽니다. 통합편: 『밤하늘 별자리 대축제』

  3. 곤충 친구들 🐝
  꿀벌이, 나비, 무당벌레, 반딧불이, 사슴벌레, 잠자리, 메뚜기, 달팽이, 거미(곤충 아니지만 반전 캐릭터로), 애벌레 — 변태(탈바꿈), 협동, 작은 생명의 역할. 통합편: 『풀숲   
  음악회』

  4. 몸속 탐험대 ❤️
  심장이, 뇌돌이, 위장이, 허파, 뼈다귀, 백혈구 용사, 혀, 눈, 귀, 피부 — 각 권 = 기관 하나 + 건강 습관(골고루 먹기, 일찍 자기, 손 씻기). 유아 건강교육 수요가 커서 실용성 
  최고. 통합편: 『우리 몸 대모험』

  5. 텃밭 친구들 🥕
  당근이, 토마토, 브로콜리, 감자, 옥수수, 가지, 호박이, 상추, 콩이, 딸기 — 편식 극복 + 씨앗→수확 성장 과정. 식사 교육과 직결되어 부모 활용도가 높습니다. 통합편: 『텃밭  
  큰 잔치』 (수확 축제 비빔밥 만들기)

  정서·생활 (교훈 중심)

  6. 마음 마을 친구들 💛
  기쁨이, 슬픔이, 화남이, 무서움이, 부끄럼이, 질투, 심심이, 설렘이, 미안함이, 용기 — 감정 인식·표현·조절. 5~7세 정서교육의 핵심 주제이고, "단점이 장점이 되는" 기존 서사 
  공식과 완벽하게 맞습니다. 통합편: 『마음 마을 무지개 축제』 (모든 감정이 다 필요하다는 결말)

  7. 탈것 구조대 🚒
  소방차, 구급차, 경찰차, 버스, 기차, 비행기, 배, 포클레인, 우주선, 자전거 — 5~7세(특히 남아) 최고 인기 주제. 각 권마다 안전 교육(신고 번호, 안전벨트, 횡단보도)을       
  자연스럽게. 통합편: 『마을 대구조 작전』 (폭우 속 모두가 힘을 합쳐 구조 — 천둥이 4권과 연결 가능)

  8. 반짝반짝 생활 친구들 🪥
  칫솔이, 비누, 수건이, 베개, 신발이, 우산이, 휴지, 손수건, 알람시계, 젓가락 — 양치, 손 씻기, 일찍 자기, 정리정돈 등 생활 습관을 물건의 시점에서. "잊혀진 물건의 서운함 →
  아이가 다시 찾아줌" 같은 서사가 잘 나옵니다. 통합편: 『우리 집 물건 운동회』

  사회·문화 (지식 확장)

  9. 우리 동네 직업 친구들 👩‍🚒
  소방관, 의사 선생님, 제빵사, 농부, 우체부, 경찰관, 선생님, 환경미화원, 사육사, 어부 — "보이지 않는 곳에서 마을을 지키는 사람들". 하현달 4권(새벽 일꾼들)의 정서와      
  이어집니다. 통합편: 『고마워요, 우리 마을』 (마을 축제에서 모든 직업이 한자리에)

  10. 우리나라 친구들 🇰🇷
  한복이, 김치, 송편이, 태권 호랑이, 한옥이, 부채, 윷가락, 탈춤이, 가야금, 연이 — 전통문화 + 명절(설날, 추석, 단오). 보름달 3권의 추석 장면과 세계관 연결. 해외 한국어   
  교육용 수요도 있습니다. 통합편: 『설날 큰 잔치』
