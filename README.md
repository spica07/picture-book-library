## 활용 방법
1. 스토리 파일의 본문으로 그림책 텍스트를 구성하세요.
2. illustration_guide.md의 **캐릭터 시트 프롬프트를 먼저 생성**해 레퍼런스로 고정한 뒤, 장면별 프롬프트로 삽화를 만드세요.
3. 글자는 이미지에 넣지 말고(프롬프트에 `no text` 유지), 편집 단계에서 한글 텍스트를 얹으세요.
